# 🎯 تطبيق متابع العادات الذكي 2026 | Habit Tracker Pro

تطبيق ويب (Web App) بسيط وقوي لمساعدتك على بناء عادات إيجابية والالتزام بها يومياً. يتميز بواجهة مستخدم أنيقة تدعم الوضع الليلي، ومؤقت تركيز، ونظام إحصائيات دقيق.



---

## ✨ المميزات الرئيسية
* **👤 بروفايل شخصي:** إمكانية إضافة اسمك، صورتك، واقتباسة تحفيزية خاصة بك.
* **🌙 وضع ليلي (Dark Mode):** واجهة مريحة للعين للتبديل بين الوضعين الفاتح والمظلم.
* **📅 تاريخ ميلادي وهجري:** دعم كامل لضبط التاريخ يدوياً بصيغة (YYYY-MM-DD) مع خانة للتاريخ الهجري.
* **⏱️ مؤقت بومودورو (Timer):** مؤقت تركيز مدمج (25 دقيقة) مع **تنبيه صوتي** عند الانتهاء.
* **📊 إحصائيات ذكية:** حساب تلقائي لنسبة الإنجاز اليومية والشهرية مع شريط تقدم (Progress Bar).
* **🚀 رسائل تحفيزية:** نظام رسائل ذكي يتغير بناءً على مستوى إنجازك الفعلي.
* **💾 حفظ البيانات:** يتم حفظ جميع عاداتك وبياناتك في المتصفح تلقائياً (LocalStorage)، مما يعني أنها لا تضيع عند إغلاق الصفحة.

---

## 🛠️ التقنيات المستخدمة
* **HTML5** - للهيكلة الأساسية.
* **CSS3** - للتصميم المتجاوب والوضع الليلي.
* **JavaScript (Vanilla)** - للبرمجة، الحسابات، وحفظ البيانات.
* **Font Awesome** - للأيقونات الاحترافية.

---

## 🚀 كيف تبدأ؟
1. قم بتحميل الملف `index.html`.
2. افتحه في أي متصفح (Chrome, Safari, Firefox).
3. أضف عاداتك وابدأ رحلة التغيير!
